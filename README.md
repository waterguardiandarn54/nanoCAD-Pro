DWG is the engineering world's shared language. nanoCAD Pro reads and writes the same format natively — no import translation, no schema conversion, no feature loss at the boundary. Command-line familiarity from AutoCAD transfers directly: the same commands, aliases and LISP routines run unchanged.

### 2D drafting tools

```
Line · Polyline · Arc · Circle · Ellipse · Spline · Hatch · Dimension
Chamfer · Fillet · Trim · Extend · Offset · Mirror · Array (rect / polar / path)
Block · External reference (XRef) · Attribute · Wipeout · Viewport
Annotative scale · Layer filter · Named UCS · Paper space / Model space
```

### 3D modelling (Pro)

Solid primitives · Boolean operations (union, subtract, intersect) · Extrude · Revolve · Sweep · Loft · Shell · Press/Pull · 3D Array · Section plane · Interference check · ACIS kernel

### Customisation API

| Interface | Language |
|---|---|
| Classic command line | nanoCAD LISP (AutoLISP-compatible) |
| Plugin API | C++ (nanoCAD SDK) |
| Managed API | .NET (C# / VB.NET) |
| Script | DCL dialog definition |

### BIM module (optional)

IFC import/export · Architectural objects (wall, floor, door, window, stair) · Object properties and attributes · BIM level 1/2 workflow
